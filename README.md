# PONG on FPGA

## FPGA MODEL
CYCLONE III board (Alera EP3C16F484C6)

## Features
- fully functional PONG game on a FPGA utilizing buttons for controls and the on-board VGA port for video output
- Clock divider for 60 Hz refresh-rate
- Buttons to move each paddle up and down
- Score tracked on the on-board HEX display
